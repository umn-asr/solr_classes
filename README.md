# Solr Classes


## Usage

### Locally

- `./script/server`
- Load data into Solr with `./script/populate [filename]`
  - `./script/populate examples/courses.json`, for example

## Development

This repo uses [Scripts to Rule Them All](http://githubengineering.com/scripts-to-rule-them-all/)

### Getting Started
- Clone repo
- `./script/setup`

### Updating
- Pull repo
- `./script/update`

### Run Tests
- `./script/test`

### Running a Server
- `./script/server`

### Deployment
- `./script/deploy`
